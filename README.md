# Mini_Banking_Apex
Basic Banking System's written by C++ 


## Features
- Savings Account
- Checking Account
- Overdraft Protection
- Multi-Currency Support
- Transfer & Exchange Rate
- Transaction History
- File Save/Load
- Double Spending Demo
- Mutex Synchronization

## Technologies
- C++
- OOP
- STL
- Thread
- Mutex
- Smart Pointer

## Run

g++ main.cpp -o banking
./banking
